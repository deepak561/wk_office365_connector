    Integrating Office365 with Odoo!

    Odoo Office365 Connector is a perfect solution to flawlessly integrate the two amazing solutions that are Odoo and Office365. The module helps you to import and export all the necessary data to manage the accounting.

    Import/Export Functions for Sale Orders

    Odoo Office365 Connector facilitates you to easily import/export the sale orders. The module allows you to import as well as export the partners, categories, product, and sale order from Odoo to Office365 and vice versa.

    Import/Export Functions for Purchase Orders

    Odoo Office365 Connector facilitates you to easily import/export the Purchase orders. The module allows you to import as well as export the vendors, categories, product, and purchase order from Odoo to Office365 and vice versa.

    Import/Export Functions for Accounting

    Odoo Office365 Connector facilitates you to easily import/export the Accounting. The module allows you to import the Sale Taxes, Tax Codes, Payment Methods, Payment terms, Invoice, Payment, and Account from Office365 to Odoo. You can export accounts, invoice and payment from Odoo to Office365.
