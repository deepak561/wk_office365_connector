Importing Data to Odoo

You can import data like tasks, projects, events and contacts in bulk just by a bulk.

You can Import these data manually or via the cron scheduler.

Exporting Data to Office 365

You can export data like tasks, projects, events and contacts in bulk just by a bulk.

You can export these data manually or via the cron scheduler.

Below is the list of features Odoo Office365 Connector

Connect Office365 to Odoo Instance

  • Connect your Odoo with as Office 365 Platforms you want

Cron to Schedule the Functions

  • Cron runs to import as well as export the data in the scheduled time

Simplified Import Function

  • Common Import Button for bulk operations
  • All you need to do is select the Instance Id and Operation; moreover, you can also set the limit of importing data

Streamlined Export Function from All the Channels
